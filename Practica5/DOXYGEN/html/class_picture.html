Detailed Description

This class provides methods for manipulating individual pixels of an image. The original image can be read from a .jpg, .gif, or .png file or the user can create a blank image of a given size. This class includes methods for inputplaying the image in a window on the screen or saving it to a file.

Pixel (x, y) is column x and row y. By default, the origin (0, 0) is upper left, which is a common convention in image processing. The method setOriginLowerLeft() change the origin to the lower left.

For additional documentation, see Section 3.1 of Introduction to Programming in Java: An Interinputciplinary Approach by Robert Sedgewick and Kevin Wayne.
